# 继电保护逻辑教学系统 - 使用指南

## 系统访问

1. 打开浏览器（推荐 Chrome、Edge 或 Firefox）
2. 输入系统地址（由技术人员提供）
3. 进入登录页面

## 首次使用配置

### 配置服务器地址

首次使用时，需要配置后端服务器地址：

1. 在登录页面，点击右上角的 **⚙**（齿轮）按钮
2. 在弹出的设置窗口中，输入服务器地址
   - 格式：`IP地址:端口号`
   - 例如：`192.168.1.100:8080`
3. 点击 **测试连接** 按钮
   - 成功：显示"连接成功"
   - 失败：检查地址是否正确，或联系技术人员
4. 点击 **保存** 按钮
5. 返回登录页面，输入账号密码登录

### 修改服务器地址

如需修改服务器地址：

1. 退出当前账号（点击右上角"退出登录"）
2. 在登录页面重新配置服务器地址

## 登录账号

系统提供三种角色账号：

| 角色 | 用户名 | 初始密码 | 用途 |
|------|--------|----------|------|
| 学员 | student | 123456 | 学习和练习 |
| 教师 | teacher | 123456 | 教学管理 |
| 管理员 | admin | 123456 | 系统管理 |

**⚠️ 重要提示**：首次登录后请立即修改默认密码！

## 修改密码

1. 登录系统后，点击右上角的用户名
2. 选择"修改密码"
3. 输入当前密码和新密码
4. 点击"确认修改"

## 常见问题

### 1. 提示"请先配置服务器地址"

**原因**：未配置后端服务器地址  
**解决**：点击右上角 ⚙ 按钮，配置服务器地址并保存

### 2. 连接测试失败

**可能原因**：
- 服务器地址输入错误
- 后端服务未启动
- 网络不通

**解决方法**：
- 检查地址格式是否正确（IP:端口）
- 确认后端服务正在运行
- 联系技术人员检查网络

### 3. 登录失败

**可能原因**：
- 用户名或密码错误
- 账号被禁用

**解决方法**：
- 确认用户名和密码输入正确（注意大小写）
- 如忘记密码，联系管理员重置

### 4. 图片无法显示

**可能原因**：
- 服务器配置问题
- 网络问题

**解决方法**：
- 刷新页面重试
- 联系技术人员检查服务器

### 5. 页面加载缓慢

**可能原因**：
- 网络带宽不足
- 服务器负载过高

**解决方法**：
- 检查网络连接
- 避开使用高峰期
- 联系技术人员优化
